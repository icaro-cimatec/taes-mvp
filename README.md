
# Plataforma de Artesãos (MVP - Sprint 1)

Este projeto é o MVP de uma plataforma que conecta artesãos com clientes. A primeira sprint inclui:

- Cadastro de artesãos
- Login com JWT
- Cadastro de produtos
- Catálogo público de produtos

## Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/icaro-cimatec/taes-mvp
cd taes-mvp
```

### 2. Instale as dependências

```bash
pip install -r requirements.txt
```

### 4. Rode o backend

```bash
python app.py
```

### 5. Acesse o frontend

Abra `index.html` no navegador.

## Tecnologias

- Python + Flask
- SQLite + SQLAlchemy
- HTML + JS
